# Forsaken Bot — Commands

## 🎫 Tickets
- `/ticketpanel` — Send the ticket panel *(Manage Server)*
- `/closeticket` — Close the current ticket *(Founder)*
- `/adduser <member>` — Add a user to a ticket *(Staff)*
- `/removeuser <member>` — Remove a user from a ticket *(Staff)*

## 🔨 Moderation
- `/warn <member> <reason>` — Warn a member *(Staff)*
- `/warnings <member>` — View a member's warnings *(Staff)*
- `/clearwarnings <member>` — Clear all warnings *(Founder)*
- `/announce <channel>` — Send a formatted announcement *(Staff)*
- `/embed <channel>` — Send a custom embed *(Staff)*
- `/say <channel> <message>` — Send a message as the bot *(Staff)*

## ✅ Verification
- `/setupverification` — Deploy the verification panel *(Founder)*

## 🔧 Utility
- `/userinfo [member]` — Show member info *(Everyone)*
- `/serverinfo` — Show server info *(Everyone)*
